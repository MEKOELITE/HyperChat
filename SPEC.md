# HyperChat - 高情商对话辅助系统

## 1. 项目概述
- **项目名称**: HyperChat
- **类型**: Android 应用
- **核心功能**: 基于AI的聊天策略助手，为用户在社交场景中提供实时策略建议与话术生成

## 2. 技术栈
- **语言**: Kotlin
- **UI框架**: Jetpack Compose
- **架构**: MVVM + Clean Architecture
- **网络**: Retrofit + OkHttp
- **本地存储**: Room
- **OCR**: ML Kit Text Recognition
- **依赖注入**: Hilt
- **异步**: Kotlin Coroutines + Flow

## 3. 功能范围 (MVP完整版)

### 3.1 角色与目标建模
- 对方身份选择：同事/上级/下属/客户/朋友/暧昧对象/恋人/家人/陌生人
- 关系状态：熟悉程度滑块、关系温度滑块、历史矛盾选项
- 聊天目标：拉近关系/邀请吃饭/请求帮助/推销产品/挽回关系/拒绝请求
- 聊天风格：真诚型/幽默型/成熟稳重/轻松自然/商务专业

### 3.2 实时聊天辅助
- 输入对方回复
- 显示AI分析（语气与意图）
- 提供3种回复方案（理解型/轻松型/目标推进型）

### 3.3 截图解析
- 图片上传（相册/拍照）
- OCR文字识别
- 聊天结构解析
- 分析报告：兴趣程度/情绪状态/防御程度/关系趋势
- 回复建议

### 3.4 对话策略系统
- 多轮策略展示
- 当前阶段指示
- 推荐话术

### 3.5 高情商库
- 场景分类：恋爱聊天/职场沟通/客户销售/冲突化解/社交破冰
- 典型对话示例
- 高/低情商表达对比

### 3.6 聊天复盘
- 聊天记录导入（文字输入）
- 社交分析报告：情绪曲线/对方投入度/关键转折点
- 改进建议

## 4. UI/UX 设计方向
- 简洁现代的聊天风格
- 卡片式布局展示建议
- 底部导航：首页/聊天/截图/复盘/我的
- 主题色：蓝色系 (#2196F3主色)
- Material Design 3

## 5. 数据层设计
- Room本地存储会话历史
- DataStore存储用户设置
- Repository模式管理数据
